# 🚀 Sobre o projeto

O **Sales Analysis System** é um sistema desenvolvido em Python com foco em:

* análise de vendas
* manipulação de dados
* automação simples
* lógica de programação
* organização de informações

O projeto simula um pequeno sistema comercial capaz de cadastrar vendas, calcular faturamento e gerar relatórios utilizando estruturas de dados em Python.

# ⚙️ Funcionalidades

## 📌 Cadastro de vendas

Permite registrar:

* produto
* quantidade
* preço

---

## 📌 Relatório de vendas

Exibe todas as vendas cadastradas no sistema.

---

## 📌 Cálculo de faturamento

Calcula automaticamente o faturamento total com base nas vendas registradas.

---

## 📌 Produto mais vendido

Mostra o produto com maior quantidade vendida.

---

## 📌 Exportação CSV

Salva os dados em arquivo:

```text
vendas.csv
```

---

## 📌 Leitura de CSV

Lê os dados salvos no arquivo CSV.
